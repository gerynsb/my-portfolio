Generate Favicon

Upload logo.png di folder public, lalu gunakan tool online:

  • Favicon.io - Favicon Converter
  • RealFaviconGenerator

Yang dibutuhkan:

  • favicon.ico (16x16, 32x32)
  • icon-32.png (32x32)
  • icon-192.png (192x192)
  • apple-icon.png (180x180)

Instruksi:

  1. Simpan logo DG yang sudah dikirim sebagai 'logo.png' di folder public/
  2. Buka https://favicon.io/favicon-converter/
  3. Upload logo.png
  4. Download hasil generate
  5. Rename file sesuai kebutuhan di atas
  6. Copy ke folder public/
